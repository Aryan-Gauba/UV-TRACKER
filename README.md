# UV Shield Tracker

A Node.js application that tells users the current UV Index and provides sun protection advice based on their location.

## How to Run
1. Clone this repository.
2. Run `npm install` to install dependencies.
3. Open `index.js` and add your [OpenUV API Key](https://www.openuv.io/).
4. Start the server with `nodemon index.js` or `node index.js`.
5. Visit `http://localhost:3000` in your browser.
